# 建云宝
---

### 文件结构

- Classes
	分为`JYBMain`、`JYBResource`、`JYBVender`
	
	`JYBVender`:放置第三方库
	
	`JYBResource`:放置资源文件，如：图片，音频，plist等
	
	`JYBMain`:包含`JYBController`、`JYBIM`、`JYBOther`
	
	- `JYBController`:建云宝相关界面分类
	- `JYBIM`:建云宝即时通讯
	- `JYBOther`:建云宝其他配置相关文件

> 具体见工程
